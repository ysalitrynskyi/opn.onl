# opn.onl — Design Context

## Users
Developers, indie hackers and privacy-conscious self-hosters who want a URL shortener
they fully control. They evaluate the marketing site on desktop while comparing tools,
then live in the dashboard during work to create, protect and measure links.

## Brand Personality
Three words: **engineered, candid, quietly confident.** It's a serious open-source tool
(Rust core, AGPL-3.0) that respects the user — no growth-hack dark patterns, no upsell,
no tracking. The voice is plain and technical, not breathless marketing.

## Aesthetic Direction
**Editorial-technical, light theme.** Warm-cool neutral paper + near-black ink + a single
deep cobalt accent. Distinctive grotesque display type with a calm humanist body; monospace
used only where it's meaningful (URLs, codes, terminal). Strong typographic hierarchy,
hairline borders, intentional asymmetric rhythm, restraint.

- **Theme:** light (trust + readability for a broad self-hoster audience; harder and more
  memorable than another dark "dev tool").
- **Type:** Bricolage Grotesque (display/headings) + Hanken Grotesk (body/UI) + JetBrains Mono (code/URLs).
- **Color:** `primary` = deep cobalt (oklch, replaces the old washed sky-blue). Neutrals
  tinted slightly toward the cobalt hue. Green/red/amber reserved strictly for state.
- **Anti-references (what it must NOT be):** generic AI-SaaS template — floating gradient
  orbs, gradient-filled text, rainbow icon tiles, purple→blue gradients, glassmorphism,
  everything centered, flat type hierarchy, decorative sparklines, colored side-stripe borders.

## Design Principles
1. **Type carries the design.** Big confident headings, clear hierarchy (≥1.25 ratio), solid color — never gradient fill.
2. **One accent, used rarely.** Cobalt is ~10% of the page; its scarcity is its power.
3. **Hairlines over shadows.** Prefer 1px borders and structure to heavy drop shadows and cards-in-cards.
4. **Show the product, honestly.** Real UI previews and real commands over decorative abstraction.
5. **Earn every element.** No element that only exists to look "designed." If it doesn't inform or guide, cut it.
